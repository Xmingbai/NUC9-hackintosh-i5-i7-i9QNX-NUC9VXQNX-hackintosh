# NUC9i5/i7/i9QNX、NUC9VXQNX  MacOS 完善程度

 核显支持核显HDMI/Tyepc-c显示输出

 前后3.5音频输出皆正常

 睡眠唤醒正常，支持USB唤醒

 雷电支持热插拔，可支持接显示器、接雷电硬盘

 板载intel AX200  蓝牙 WIFi可以驱动，网速非常不错 ，但不支持隔空随航，要想实现隔空随航附加功能 需要买一张NVME M.2 SSD转接黑果无线网卡
